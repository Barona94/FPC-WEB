# 📚 Comandos básicos de Git

A continuación se describen algunos comandos básicos de Git, junto con su definición y un ejemplo de uso.

---

## 🔹 git --version

**Definición:** Muestra la versión de Git instalada en el sistema.

**Ejemplo:**

```bash
git --version
```

---

## 🔹 git init

**Definición:** Inicializa un repositorio Git en la carpeta actual.

**Ejemplo:**

```bash
git init
```

---

## 🔹 git add

**Definición:** Agrega archivos al área de preparación , dejándolos listos para ser confirmados.

**Ejemplo:**

```bash
git add .
```

*(Agrega todos los archivos del proyecto)*

---

## 🔹 git commit

**Definición:** Registra los cambios en el repositorio junto con un mensaje descriptivo.

**Ejemplo:**

```bash
git commit -m "Primer commit"
```

---

## 🔹 git status

**Definición:** Muestra el estado actual del repositorio (archivos modificados, etc).

**Ejemplo:**

```bash
git status
```

---

## 🚀 Flujo básico de trabajo en Git

```bash
git add .
git commit -m "mensaje"
git status
```

Este flujo permite:

* Agregar cambios al proyecto
* Guardarlos en el repositorio
* Verificar el estado actual

---

## ✅ Conclusión

Estos comandos son fundamentales para trabajar con Git y permiten llevar un control de versiones eficiente en cualquier proyecto de desarrollo.
